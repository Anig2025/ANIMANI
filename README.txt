# My Portfolio Website

Welcome to my personal portfolio website! This site showcases my web development projects and provides a little about me.

## Features

- **About Me**: A brief introduction to who I am and what I do.
- **Projects**: A display of some of my notable web development projects.
- **Contact**: A simple form to get in touch with me.

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run Locally

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/my-website.git
    ```
2. Navigate to the project directory:
    ```bash
    cd my-website
    ```
3. Open `index.html` in your preferred web browser.
